# 🐍 Snake Game

A classic Snake game built from scratch using vanilla HTML, CSS, and JavaScript. No frameworks, no libraries — just clean browser code.

**[▶ Play it live](https://ajunabia228.github.io/snake-game/)**

---

## Gameplay

Use the arrow keys to guide the snake. Eat the red food to grow and score points. Don't hit the walls or yourself!

| Key | Action |
|-----|--------|
| `↑ ↓ ← →` | Move snake |
| `Space` | Pause / Resume |

## Features

- Start, Pause, and Replay buttons
- Score tracking
- Game over screen
- Collision detection (walls + self)

## Run locally

```bash
git clone https://github.com/ajunabia228/snake-game.git
cd snake-game
```

Then open `index.html` in your browser, or use the **Live Server** extension in VS Code.

## Built with

- HTML5 Canvas
- CSS3
- Vanilla JavaScript
